# Installation

Elwood is [available on PyPI](https://pypi.org/project/elwood/). Compatible with Python versions >= 3.8
```
pip install elwood
```


Alternatively you can install for local development.

```
git clone git@github.com:jataware/elwood.git
cd elwood
pip install .
```
